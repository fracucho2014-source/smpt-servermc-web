  Versión: 1.20.1 Forge

¡Gracias por descargar los mods de SMPT 10-17!

INSTRUCCIONES DE INSTALACIÓN
-----------------------------

1. Asegúrate de tener Minecraft Java Edition 1.20.1
   con Forge (hasta la versión 47.4.20) instalado.

2. Localiza la carpeta "mods" de tu perfil de Forge:
   - Windows: %appdata%\.minecraft\mods
   - macOS: ~/Library/Application Support/minecraft/mods
   - Linux: ~/.minecraft/mods

3. Copia TODOS los archivos .jar de esta carpeta
   dentro de la carpeta "mods" de tu perfil de Forge.

4. Inicia Minecraft con el perfil de Forge y conéctate
   al servidor usando la IP: potential-divx.gl.joinmc.link

RECOMENDACIÓN: Se recomienda usar la Herramienta de
instalación de Minecraft para configurar todo automáticamente.
